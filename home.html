Tentang Saya

Ditulis oleh Nur Lickha Lavenia pada 03 februari 2021

Nama saya Nur Lickha Lavenia. Waktu saya kecil sampai menginjak SMP teman-teman saya memanggil saya dengan sebutan Nia, namun sejak saya masuk SMK hingga perguruan tinggi teman-teman memanggil saya dengan sebutan Lickha. Saya lahir di Kota Lamongan pada tanggal 13 juli 2001. Saya memiliki beberapa hobi, contohnya waktu kecil saya hobi sekali menggambar dan bercita-cita ingin menjadi seorang desainer gaun. Namun setelah tumbuh dewasa saya mulai bosan sehingga saya mencari hobi baru. Saat SMP, saya menemukan hobi baru yaitu memotret dan editing baik foto maupun video. Karena saya berkeinginan mengembangkan hobi saya akhirnya saya memutuskan untuk masuk SMK dan mengambil jurusan multimedia. Setelah lulus dari SMK, saya sebenarnya tidak ada keinginan untuk kuliah dan lebih memilih untuk bekerja. Namun, setelah saya pikir-pikir saat ini pendidikan sangat penting di dalam dunia pekerjaan. Akhirnya saya memutuskan untuk ikut SBMPTN dan akhirnya saya diterima di salah satu PTN yaitu UPN "Veteran" Jawa Timur dengan jurusan Sistem Informasi. Alasan saya mengambil jurusan ini karena saya ingin belajar mengenai IT dan manajemen lebih dalam lagi.
